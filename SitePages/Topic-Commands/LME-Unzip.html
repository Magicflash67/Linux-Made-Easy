  • How to Know if you have the Unzip command
      The fastest way to test if you have the unzip command is by trying to run it. To do this type user-$ unzip if an error comes up you don't have it, but if you get use cases you have the command
  • Linux How to Install the command
    • • Debian / Ubuntu – sudo apt update && sudo apt install unzip
    • • RHEL / CentOS 7 – sudo yum install unzip
    • • RHEL / CentOS 8+ / Fedora – sudo dnf install unzip
    • • Arch Linux – sudo pacman -Sy unzip
    • • openSUSE – sudo zypper install unzip
    • • Alpine Linux – doas apk add unzip
    • • Gentoo – sudo emerge --ask app-arch/unzip
    • • Slackware – sudo slackpkg update && sudo slackpkg install unzip
    • • Void Linux – sudo xbps-install -S unzip
    • • Java Package Manager Supplement (Under Development) – sudo JPM-install unzip

  • How to Run the Unzip Command
    • • 1. Confirm you have the unzip command or install it via JPM or your distro's package manager
    • • 2. Type unzip into terminal and add a space
    • • 3. If on GUI drag your .zip file to terminal, or manually type the file path
    • • 3.5 WARNING TIP: if a file or folder has a space in it, wrap the entire path in quotes
    • • 4. Press enter and the files will extract into your home directory
    How to Run the Unzip Command into a Custom Directory
    • • 1. Confirm you have the unzip command or install it via JPM or your distro's package manager
    • • 2. Type unzip into terminal and add a space
    • • 3. Drag the .zip file to terminal or type the file path
    • • 4. Add -d after the file path
    • • 5. Drag or type your custom directory path
    • • 6. Press enter and the files will extract into the specified directory
